## Requirements
To run this project, install it locally using npm:

- git clone git@github.com:suhassalian27/E-Commerce-Website-using-ReactJS-NodeJS.git
```
  $ cd E-Commerce-Website-using-ReactJS-NodeJS
```
- To Run Backend
  - open terminal 
```
  $ npm install
  $ npm start
```
- To Run Frontend
  - open new terminal
```
  $ cd frontend
  $ npm install
  $ npm start
```
